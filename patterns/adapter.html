Adapter Pattern

Concept

The Adapter is a structural design pattern that allows objects with incompatible interfaces to work together. It acts as a bridge that translates one interface into another.

Example

class Target {
  request() {
    return "Target: the default request.";
  }
}

class Adaptee {
  specificRequest() {
    return "Adaptee: a specific request.";
  }
}

class Adapter extends Target {
  constructor(adaptee) {
    super();
    this.adaptee = adaptee;
  }

  request() {
    return `Adapter: (converted) ${this.adaptee.specificRequest()}`;
  }
}
