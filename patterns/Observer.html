Observer Pattern - Web Project Guide

1. What is Observer?

The Observer pattern allows objects (subjects) to notify other objects (observers) about changes in their state automatically, without tight coupling. Common uses in web projects include:

  • Synchronizing UI with data models
  • Reacting to user events
  • Notifying components of state changes

2. When to Use

  • Multiple components depend on a single data source
  • Dynamic reaction to state changes is needed
  • Reducing tight coupling between components

Examples: live chat, notifications, data subscriptions, user event tracking.

3. Implementation Checklist

  • Define a Subject – the object whose state will be observed
  • Create Observers – objects that react to state changes
  • Subject methods: attach(observer), detach(observer), notify()
  • Each observer implements update(data) method
  • Ensure the Subject does not directly call observer logic

4. Pros & Cons

Reduces coupling between components
Easy to add new observers
Suitable for dynamic systems with multiple dependent elements
Testable: observers can be mocked
Hard to track update order
Many observers → performance cost
Risk of memory leaks if observers are not detached
Debugging can be complex with many interdependent observers

5. Alternatives

  • Mediator: central object manages communication between components
  • Event Bus / Pub-Sub: asynchronous messaging, less coupling
  • Two-way data binding: automatic UI update on model change (Vue.js / Angular)
  • Redux / MobX / Zustand: global state with reactive updates

6. Example

// Subject class DataStore { constructor() { this.observers = []; this.data = null; } attach(observer) { this.observers.push(observer); } detach(observer) { this.observers = this.observers.filter(obs => obs !== observer); } notify() { this.observers.forEach(obs => obs.update(this.data)); } setData(newData) { this.data = newData; this.notify(); } } // Observer class UIComponent { update(data) { console.log("UI updated with", data); } } const store = new DataStore(); const component = new UIComponent(); store.attach(component); store.setData({ user: "Leon" }); // "UI updated with { user: 'Leon' }"
